# 🎬 QuickShow - Movie Ticket Booking Application

QuickShow is a full-stack Movie Ticket Booking Application built using the MERN Stack. The platform allows users to browse movies, view show details, select seats, and book tickets seamlessly. It also provides an admin dashboard to manage movies, shows, and bookings.

## 🌐 Live Demo

**Live Website:** https://movie-ticket-frontend-seven.vercel.app/

## 🚀 Features

### User Features

* Browse currently available movies
* View movie details, cast, ratings, and trailers
* Select show timings
* Interactive seat selection
* Real-time seat availability checking
* Secure user authentication with Clerk
* Online ticket booking
* Booking history management
* Responsive design for desktop and mobile devices

### Admin Features

* Add and manage movies
* Create and manage shows
* Monitor bookings
* Manage seat availability
* Dashboard for administration

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### Authentication

* Clerk Authentication

### APIs & Services

* TMDB API (Movie Data)
* Inngest (Background Jobs)

### Deployment

* Frontend: Vercel
* Backend: Render / Node Server
